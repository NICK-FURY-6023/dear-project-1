## 🚀 **How to Run:**

### **Linux/Mac:**
```bash

source venv/bin/activate
python manage.py runserver 0.0.0.0:8000
```

### **Windows:**
```cmd
setup.bat
venv\Scripts\activate
python manage.py runserver 0.0.0.0:8000
```

### **Direct Run (if already set up):**
```bash
python manage.py runserver 0.0.0.0:8000
```

Then open: **http://localhost:8000**
